# 2ides Chess

A professional, modern chess experience built with Next.js, Firebase, and Genkit.

## How to Sync with GitHub

To connect this project to your GitHub account and keep it updated:

1. **Locate the Project Name**: At the very top of the Firebase Studio interface (top-left or top-center), you will see the name of your project (e.g., `studio-xxx`).
2. **Open the Menu**: Click on that **Project Name**.
3. **Connect to GitHub**: From the dropdown menu that appears, select **"Connect to GitHub"**.
4. **Follow Prompts**: Link your GitHub account and choose to create a new repository.
5. **Syncing**: Once connected, every time you click the **"Publish"** button in the top-right corner, your GitHub repository will be automatically updated with the latest code changes.

## Tech Stack
- **Framework**: Next.js 15
- **Database/Auth**: Firebase (Firestore & Auth)
- **AI**: Genkit with Google AI (Gemini)
- **Styling**: Tailwind CSS & ShadCN UI

## Features
- **PWA Ready**: Installable on smartphones for a native app experience.
- **Rule Protection**: The King cannot be captured (following official chess rules).
- **Pro Design**: Clean interface with subtle watermarks and 2ides branding.
