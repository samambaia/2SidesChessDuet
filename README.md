# 2ides Chess

A professional, modern chess experience built with Next.js, Firebase, and Genkit.

## How to Sync with GitHub

To connect this project to your GitHub account and keep it updated:

1. Click on the **Project Name** or the **Publish/Deploy** button at the top of the Firebase Studio interface.
2. Select **"Connect to GitHub"** from the dropdown menu.
3. Follow the authorization prompts to link your GitHub account.
4. Choose to create a new repository or link to an existing one.
5. Once connected, every time you click **"Publish"**, your GitHub repository will be automatically updated with the latest code.

## Tech Stack
- **Framework**: Next.js 15
- **Database/Auth**: Firebase (Firestore & Auth)
- **AI**: Genkit with Google AI (Gemini)
- **Styling**: Tailwind CSS & ShadCN UI
