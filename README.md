# 2ides Chess

A professional, modern chess experience built with Next.js, Firebase, and Genkit.

## How to Sync with GitHub

If clicking the project name at the top opens the console instead of a menu, please try these steps:

### Option A: Use the Menu (Precise Click)
1. Look at the top-left breadcrumb: `Firebase Studio > ChessDuet`.
2. Click exactly on the **Chevron symbol (`>`)** or the word **"ChessDuet"**.
3. A dropdown should appear. Select **"Connect to GitHub"**.

### Option B: Use the Command Palette (Failsafe)
1. Press `Ctrl + Shift + P` (Windows/Linux) or `Cmd + Shift + P` (Mac).
2. Type **"GitHub"** in the search bar that appears at the top.
3. Select **"GitHub: Connect to GitHub"** from the list.

### Option C: Sidebar Icon
1. Look at the vertical bar on the far left of the screen (where the Explorer/Folder icon is).
2. Look for a **GitHub icon** or a **Source Control icon** (three circles connected by lines).
3. Click it to find the initialization/connection options.

## Features
- **PWA Ready**: Installable on smartphones for a native app experience.
- **Rule Protection**: The King cannot be captured (following official chess rules).
- **2ides Branding**: Clean interface with professional watermark and styling.
- **English Localization**: Fully translated for international play.
